# SQL Basics – Levels 1–4

This repository contains a set of beginner SQL exercises grouped by difficulty level.
The goal is to practice common SQL operations such as filtering, sorting, joins, and aggregations.

### Structure

- **Level 1 – Selects**
  Basic SELECT queries, filtering and sorting data.

- **Level 2 – Joins**
  Working with relationships between tables.

- **Level 3 – Aggregations**
  Using GROUP BY, COUNT, SUM and simple analytical tasks.

- **Level 4 – Advanced**
  More complex queries such as rankings and conditional analysis.

Each file includes one task with a short description and the SQL query.
The exercises are based on simple sample datasets and can be run in BigQuery or any SQL environment.

More tasks will be added over time.
